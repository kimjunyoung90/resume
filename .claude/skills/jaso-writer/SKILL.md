# jaso-writer — 설득형 글 작성·퇴고 스킬

사용자(김준영)의 글쓰기 기준을 그대로 적용해 자소서·지원동기·경력기술서를 쓰고 다듬는다.
핵심 철학: **"사실·논리·독자"를 기준으로, 본질(메시지)을 먼저 잡고 단어 하나까지 깎아낸다. 과장으로 부풀리지 않고, 정직하게 쓰되 경험으로 보여줘서 설득한다."**

## 작업 프로세스 (순서 엄수)

1. **메시지 먼저 확정** — 문장을 만지기 전에 "이 경험/문단에서 *뭘 말하고 싶은가*"를 먼저 정한다.
   - "세 가지를 했다" 같은 나열이 아니라 *하나의 핵심 역량/통찰*이 무엇인지 묻고 합의한다.
   - 예: 1번=분석력, 2번=자동 복구, 3번=대용량 버티기, 4번=부채 개선 태도.
2. **사실 검증** — 모든 수치·주장을 원본(경력기술서·트러블슈팅 문서)과 대조한다.
   - 지어내지 않는다. 확인 안 되면 사용자에게 묻는다. (resume 메모리 feedback_resume_defensible)
   - 실제로 한 게 아니면 프레이밍하지 않는다(예: 신뢰성 작업을 "부하 견디기"로 둔갑 금지).
3. **구조 배치** — 아래 "구조 원칙"에 따라 배치한다.
4. **문장 다듬기** — 아래 "문장 지침"을 한 문장씩 적용한다.
5. **사용자와 수렴** — 한 번에 확정하지 말고, 선택지를 제시하거나 의도를 되물어 사용자가 방향을 잡게 한다(소크라테스식). 제안을 강요하지 않는다.

## 구조 원칙

- **두괄식**: 결론(핵심 주장/성과)을 먼저, 근거를 뒤에.
- **설득 원리 "상대의 필요 → 나의 부합"**: 채용사가 필요로 하는 인재상(JD·우대·인재상)을 세우고 거기에 경험·수치로 부합을 입증.
- **적합성 선언은 서론과 마무리에서만 (수미상관)**. 본론은 적합 선언 없이 경험·근거만 담아 **telling이 아니라 showing**으로 읽는 사람이 스스로 판단하게 한다.
- **본론 문단**: 도입(상대의 필요)이 군더더기면 빼고 **경험(역량)부터** 시작. 단계는 "행동 먼저 → 결과" 패턴, 필요하면 첫째·둘째·셋째.
- **하나의 주장에 핵심 경험 하나**. 메시지와 결이 다른 부가 사례는 뺀다.

## 문장 지침 (사용자가 누적해 확정한 규칙)

1. **사실확인·과장 금지** — 검증 불가 일반화("성능 문제가 잦다", "한 번의 지연이 곧 이탈", "X가 필요하다(필수)") 쓰지 않는다. 단정 대신 조건부("방치하면")로. 수치는 실제 값만.
2. **논리적 인과 일치** — 앞뒤가 성립해야 한다(예: 추측→"엉뚱한 곳 탐색에 시간 낭비"가 맞지 "엉뚱한 곳을 고친다"는 틀림 / "응집도는 회복이 아니라 개선·높임").
3. **무엇을 했는지(행동) 먼저, 명확히** — "의심했다"처럼 생각만 적지 말고 한 행동을 진술. 행동과 결과를 분리.
4. **긴 쉼표 연결 문장은 분리** — 가독성 해치면 여러 문장으로.
5. **배경 모르는 독자도 이해** — 맥락(왜 그 상황인지)을 짧게 깔고, 적용 대상(어느 기능/작업)을 먼저 밝힌다. 갑자기 용어·사례가 튀어나오지 않게.
6. **공식 문서체** — 인사담당자·실무자가 보는 글. 구어·어색한 표현 지양("가렸습니다", "엉뚱한 곳", "채운 경험" 등).
7. **부연·중복 금지** — 같은 말 반복 문장은 합치거나 삭제(예: "중요합니다"+"필요합니다").
8. **단어 정밀성** — 의미가 미세하게 어긋나면 고친다("응답을 단축"→"응답 시간을 단축", "보장"→"복구").
9. **도구·구현 디테일 절제** — 깊은 구현(특정 라이브러리, 이중 캐시 내부 구조 등)은 본문에서 빼고 면접 카드로. 단, 통용 용어(Cache Stampede, Outbox, 멱등 등)는 역량 신호로 남길 수 있다.
10. **상투구 금지** — "빠르게 적응해 기여하겠습니다" 류 클리셰 대신, 이미 해낸 근거로 확신을 준다.
11. **em-dash("—") 사용 자제** — 괄호나 줄 분리로. (단, 항목 나열 대시 "역량 — 설명"은 사용자가 허용한 맥락에선 가능)

## 검증 체크리스트 (마무리)

- 모든 수치·주장이 원본으로 방어 가능한가?
- 검증 불가 일반화·과장이 없는가?
- 각 문장의 인과가 성립하는가?
- 배경 모르는 사람이 읽어도 맥락이 끊기지 않는가?
- 부연·중복·상투구가 없는가?
- 적합성 선언이 서론·마무리에만 있고 본론은 근거 중심인가?
- (자소서) 글자수 한도 내인가? JD 자격·우대 항목이 충분히 커버되는가?
