# Project Overview

This project is a personal resume website for Kim Junyoung, a backend developer. It is built with HTML, CSS, and a little bit of JavaScript. The resume is also available in Markdown format. The project is deployed to GitHub Pages.

# Building and Running

There is no build process for this project. To run the project, simply open the `index.html` file in a web browser.

To deploy the website to GitHub Pages, run the following command:

```bash
npm run deploy
```

# Development Conventions

There are no explicit development conventions in this project. However, the code is well-formatted and easy to read.
