MCP — 주의사항

MCP 목록 확인: /mcp

› /mcp

  jira        · ✓ connected
  confluence   · ✓ connected
  slack       · ✗ disabled

주의: 활성화된 모든 MCP가 제공하는 기능 목록과 사용법이 컨텍스트에 로드됨
→ 예: Notion 15,076 토큰 + Slack 7,961 토큰 = MCP만으로 27,000+ 토큰
→ 미사용 MCP는 항상 꺼둘 것
→ 필요한 기능만 담은 커스텀 MCP를 만들어 사용 권장
